# AIOps 平台使用指南

## 启动平台
要启动 AIOps 平台，请运行以下命令：
```bash
cd /path/to/aiops
./scripts/start.sh
```

## 查看日志
平台日志存储在`/path/to/aiops/logs`目录下，也可以通过以下命令查看：
```bash
docker logs -f aiops-backend
```

## 功能简介
AIOps 平台包含以下核心功能:
1. 智能根因分析 - 自动检测并定位系统问题根源
2. 预测性扩缩容 - 根据历史数据预测资源需求
3. 自动修复 - 自动解决常见系统问题
4. 智能小助手 - 提供问答服务和文档检索
